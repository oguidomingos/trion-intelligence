### Dados Estratégicos Extraídos

**Persona ICP:**
- Esteticista/biomédica, 28-45 anos
- Faturamento R$20-80k/mês
- Cidades pequenas/médias (GO, MG)
- Dor: agenda inconsistente, dependência de indicação

**Produto Core:**
- Trion Growth Core: R$2.500-4.000/mês
- Setup T14 (14 dias)
- Downsell: Profilaxia Digital R$1.500/mês

**Posicionamento:**
- "Especialista em crescimento previsível para clínicas de estética"
- Clínica não compra funil/IA — compra agenda cheia, status local, previsibilidade
