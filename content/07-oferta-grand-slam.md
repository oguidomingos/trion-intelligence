# Módulo 7: Oferta Grand Slam (Hormozi)

## A Equação de Valor

```
VALOR = (Resultado dos Sonhos × Probabilidade Percebida) / (Tempo de Espera × Esforço)
```

Para maximizar valor: aumente o numerador (resultado + probabilidade) e diminua o denominador (tempo + esforço).

## Aplicação para Estúdios de Tatuagem

### Tracking de Conversas como Arma Secreta

Nenhuma agência de marketing para tatuadores oferece tracking de conversas como serviço core. Todas vendem ads e posts. A Trion vende previsibilidade de receita.

### Estrutura de Value Stack (Hormozi-style) para Tatuadores

1. **Core:** o serviço de marketing em si
2. **Bônus 1:** setup completo de CRM com inbox centralizado
3. **Bônus 2:** Google Business Profile otimizado
4. **Bônus 3:** site responsivo com booking integrado
5. **Bônus 4:** programa de indicação automatizado
6. **Garantia:** "Agenda cheia em 90 dias — ou você não paga o mês 4"

### Flash Days e Promoções (Estratégia de Aquisição)

- Flash days com acesso antecipado para lista de email = urgência + lealdade
- Pacotes grupo: 5 tatuagens combinadas pelo preço de 4
- Gift cards com desconto (R$500 por R$400)
- "Artist's Choice" com desconto: cliente deixa o artista escolher → conteúdo + ticket alto

## Framework de Precificação High-Ticket

### Por Que Cobrar Caro

O Grand Slam de Hormozi diz: **cobrar 10-100x mais** e remover escalabilidade. Adicionar personalização.

| Fator | Budget | High-Ticket |
|-------|--------|------------|
| Atenção | Dividida entre 50+ clientes | Dedicada a 5-10 clientes |
| Resultado | Genérico | Customizado |
| Garantia | Nenhuma | Performance-based |
| Acesso | Email, 48h resposta | WhatsApp direto, mesmo dia |
| Relatórios | Mensais | Semanais com dashboard ao vivo |

### Modelos de Precificação (Tatuadores)

| Modelo | Quando usar | Faixa |
|--------|-------------|-------|
| Preço fixo | Tatuagens pequenas/flash | R$200-800 (mínimo definido) |
| Hora | Custom work, sessões longas | R$200-700/hora |
| Pacote multi-sessão | Sleeves, costas inteiras | R$3.000-15.000+ |
| Misto | Flash = fixo, custom = hora | Combina previsibilidade + flexibilidade |

### Estratégia de Upsell

- Consulta presencial → projeto custom multi-sessão
- Flash pequeno → sleeve completo (mockup digital)
- Sessão única → pacote de sessões com desconto
- Tatuagem → aftercare kit premium + merchandise
- Add-ons: miniaturas complementares na mesma sessão
